5.1 Forecast

 (require whatnow/forecast) package: whatnow
The whatnow/server/orecast library provides a wrapper around whatnow/server/forecast-json that re-exports the downloads as structured data.

These functions check that all fields are present in the JSON returned from Forecast; and for some fields a check is made that the field is not-null. An error is raised if these conditions are not satisfied.

Note that placeholders are not exported, nor is any assignment to a placeholder, nor are archived entities.
